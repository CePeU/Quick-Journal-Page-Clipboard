A (little) module to easily copy your journal content (from multipage or single page) to the clipboard, save it locally to a file or print it. 
In the settings you can adjust to switch your export between plain text, (Obsidian style) Markdown or HTML.
It also allows the GM to export secrets or remove hidden secrets from the export.

You find the menü entry in the control panel (three point button in the header of the journal).

You can also define your own CSS and adjust your HTML export.
Markdown export ist rendered with its own Markdown engine and not with the native Foundry one.
Markdown export supports Obsidian style Markdown to use it in your Obsidian vaults.
Callout export ist supported for Blockquotes or details tag callouts and can be choosen to your liking. 

You have a lot of cusomizing options in the way you want to export your content.

You also can export with keybinds:   

Export to clipboard:    CTRL+ALT+C
Export to file:         CTRL+ALT+F
Export to printer:      CTRL+ALT+P

Export allways takes plafe from an opened journal which needs to be placed on top of any other applikation.

If more people find the plugin usefull I might considere to add an API so more automated exports would be possible.
